# crypto.cr

**Work In Progress**

Adds new crypto features to crystal. Currently planned:

Algorithems
- AES CBC 128/192/256
- AES GCM 128/192/256

Key Stretching
- bcrypt


## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  crypto+:
    github: chris-huxtable/crypto-plus.cr
```
